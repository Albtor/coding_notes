// BOX MODEL
//All elements in html are BOXES with Content. Or content with a box/space around
//All elements are boxes. If you do nothing the content will fill the box will fill all the space, line margin collapse
Box-sizing: border-box; //If not stated, padding and border add to the size //height-width// not adding it results in more pixels
border: 1px solid black; //Border around an element, around padding.
border-width: 10px, 20px, 40px, 40px; //Top Right Bottom Left 
padding: 2px 2px 4px 4px; //Separation between the border and the content
margin: 2px 5px; //it collapses if two adjacent elements have margin. Si Uno tiene 50px de margin y Otro tiene 20,el margin que resultará de distancia entre los dos es de 50px. No sería de 80px
