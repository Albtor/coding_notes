px: Unidad de medida fija, según la resolución del monitor. No cambia

Porcentaje: depende del contenedor padre

View Width /height: depende unicamente del tamaño de la pantalla

REM - Root font size. 1= 100% of the size (same) 2=200% 0.5= 50%. Don't accumulate like em

EM - Based on the parent's font size. They can multiply so better use rem. Em

When is recommended to use em: When you want an object to have padding that scales with the size of the element. For example: padding: 0.25em; For scaling padding

Problem of using px for the font size. If someone resizes the browser, the font size will remain on the px set. With rem

Same with percentages which is similar to em regarding to fonts, as reppresents the % of a font
