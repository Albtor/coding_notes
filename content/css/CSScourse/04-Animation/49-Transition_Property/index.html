Transition Property

  • transition: all 1s ease-in-out .5s - all can be changed to any property (color, background-color. Set into the normal tag (div) not in the hover
  • transition-property: all;
  • transition-duration: 1s;
  • Specify the properties: color 1s; background-color: 2s; o all
  • transition-timing-function: linear; ease-in ease-in-out, cubic-bezier. this is editable
  • cubic bezier editor/curve: opening dev tools and on the property transition there is a box/cube next to the type. Clic in to edit
  • transition-delay: .5s; time it takes to start the transition
  • transition: all 1s ease-in-out .5s
  • Multiple properties: transition: color 1s, background-color 3s;
  • Can also be done like: transition-property: color, background-color; transition-duration: 2s;
