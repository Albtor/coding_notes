Columns

  • column-count: 2
  • column-width: 100px
  • Can combine both, but if it doesn't fit the columns' width requested it will reduce the number of columns
  • Column-count: 3 column-width: 200px;
  • column-rule: 1px solid black - Its like border but just to separate columns. Too big, it overlaps the content.
  • column-span: all. (All, none) Span elements in different columns. 1em as default
  • column-fill: auto / balance; Auto requires height. Puts elements inside only one column, not half in one column and another half in another column
