Position

  • Position: static as default.
  • relative and absolute
  • Absolute: Depends on the position context. poistion inside of the container with relative or. It positions itself anywhere in the page if there isn't any previous container with position relative. If the parent container has p:relative, then it will position absolutely inside it., will not be out. This way a "position context is created".
  • Absolute: Always stays on the same position of the visible document but not out of it (unless you set negative values). It will not go further from the view, -not the total height of doc, without scroll-. So if height is 200vh, the element will be in the visible area.
  • Relative
  • Fixed: stays positioned in the specific spot. It's like absolute. Depends on the screen.
  • Sticky: stays on its parent container
  • Values to set to define position (any): top, left, right, bottom
