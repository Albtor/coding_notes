Types of display: block, inline, inline-block, none, flex, grid

Block. takes the whole line OF ITS PARENT. Cannot have elements on its right or left. you can set hight or width. By default it takes 100% of the width of the parent and the height of the content. Examples: div

Inline: takes the minimum amount of space necessary. Can have elements on their sides but height and width are not modifiable and dependant of the content. Example: span

Inline-block: you can set height and width, width is the content so the use is better for the height. Can have elements on its right or left as opposed as just block which can't

None: makes the element not displayed on the page
