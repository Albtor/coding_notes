Calc function

  • Arithmetic in CSS
  • accepts + / * - (but addition/substraction needs to specify the units 10px + 2px)
  • margin: calc(10px * 2); 20 Px of margin in the paragraph
  • combine units in one single value
  • .p2{ --margin: 30px; margin: calc(var(--margin) * 2); } . Here we specify the variable of 30px
  • .p3{ --spacer: 20; margin: calc(var(--spacer) * 1px); } This way we don't specify the units but then multiply it for 1 pixels. BUt you can set to 1%, 1rem, etc
