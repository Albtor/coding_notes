Custom properties

  • :root{--namevariable: value;}
  • Ex: :root{--primary-color: green;}
  • h2{background-color: var(--primary-color);}
  • if not declared, you can set a default value. ex: p{margin: var(--margin, 10px);} It means that if the variable --margin is not declared in root, it will take 10px of value
  • You can override the root value setting a new value in any selector. ie: p{ --border: 3px solid yellow;}
