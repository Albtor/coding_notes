Custom Fonts

  • fallback always. font-family: Monserrat, sans-serif,
  • Google fonts: embed (link or import)
  • link + font family in css - link href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@600&display=swap" rel="stylesheet"
  • in css @import - @import url('https://fonts.googleapis.com/css2?family=Open+Sans:wght@600&display=swap'); + font-family
  • @font-face - font-family: 'Montserrat2'; src: url('fonts/fontawesome-webfont.woff2') format('woff2'), url('fonts/fontawesome-webfont.woff');
  • font-display: block, swap, fallback, optional
  • fallback: if it takes x time to download the font, it will stay on the default one
